## Client (React)

### Requirements
- Node.js 18+

### Run
```bash
npm install
npm run dev
```

App runs on `http://localhost:5173` and calls the API at `http://localhost:8080`.

### API base URL
Create a `.env` file (or use env vars) to change the API:

```bash
VITE_API_BASE=http://localhost:8080
```
